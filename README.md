**Launch My Career** was built to fix that.

## ЁЯТб Our Philosophy

### Proof Matters. Here's Why.

In today's job market, **claiming** a skill and **proving** a skill are not the same thing. Recruiters and hiring managers are trained to spot the difference.

| What Most Graduates Do | What Launch My Career Graduates Do |
|------------------------|-------------------------------------|
| List skills on a resume | Link to projects that demonstrate skills |
| Say "proficient in Python" | Show a GitHub repo with real Python work |
| Have a generic LinkedIn bio | Have a profile optimized for discovery |
| Submit resumes into the void | Get callbacks because their profile validates itself |

**Three truths that drive everything we build:**

1. **Recruiters trust what they see.** Hiring managers spend more time on visible work than reading claims on paper.
2. **Evidence beats words.** Portfolios, GitHub repos, and LinkedIn activity act as real-world proof.
3. **Standing out is a skill.** While others list skills, you'll demonstrate them тАФ making you memorable.

## ЁЯЪА Getting Started (Dev)

### Prerequisites

```bash
Node.js >= 18.x
npm >= 9.x  (or yarn / pnpm)
```

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/launch-my-career.git
cd launch-my-career
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Set Up Environment Variables

```bash
cp .env.example .env.local
```

Fill in the required values in `.env.local`:

```env
NEXT_PUBLIC_SITE_URL=http://localhost:3000
NEXT_PUBLIC_GA_ID=your_google_analytics_id
CONTACT_FORM_ENDPOINT=your_form_endpoint
```

### 4. Run the Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### 5. Build for Production

```bash
npm run build
npm start
```
